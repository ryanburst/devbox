devbox — sync WSL hosts to Windows (for browser .local URLs)
============================================================

.cmd files often do NOT offer "Run with elevated access" on corporate PCs.
Use elevated PowerShell instead:

STEP 1 — Open PowerShell WITH elevation
  • Start menu → search "PowerShell"
  • Right-click Windows PowerShell → "Run with elevated access"
    (your company menu — NOT "Run as administrator" / domain password)

STEP 2 — Paste these two lines:

  cd $env:LOCALAPPDATA\devbox
  powershell -ExecutionPolicy Bypass -File .\apply-dev-hosts.ps1

STEP 3 — Press Enter at "Press Enter to close" when you see Success.

STEP 4 — Still in that window (optional):

  ipconfig /flushdns

Then test in Edge/Chrome: http://yourname.local:PORT

---
If something fails, open this file in Notepad:

  %LOCALAPPDATA%\devbox\apply-dev-hosts.log

Re-prepare from WSL Ubuntu:  devbox setup hosts
